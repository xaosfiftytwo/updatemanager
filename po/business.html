Business applications

This is a list of recommended applications suited for businesses.

When you click the install button all listed applications are going to be installed on your system. You can use the software manager to install them separately.
Gnucash

GnuCash is personal and small-business financial-accounting software.

Designed to be easy to use, yet powerful and flexible, GnuCash allows you to track bank accounts, stocks, income and expenses. As quick and intuitive to use as a checkbook register, it is based on professional accounting principles to ensure balanced books and accurate reports.
Homepage: http://www.gnucash.org
Dolibarr

Dolibarr ERP and CRM is a modern and easy to use web software to manage your business (contacts, invoices, orders, products, stocks, agenda, emailings, etc...).

On first run, final configuration is started. You will need to fill in the following information:

  • Password for Dolibarr database owner: dolibarr
  • Superuser access: root / your_root_password
  • Dolibarr admin login : admin / admin

Replace "your_root_password" with your root password.
Homepage: http://www.dolibarr.org
Letodms

LetoDMS is an open-source, web-based document management system (DMS) written in PHP with a database backend. LetoDMS provides document meta-data, version control, security and easy access to your documents.

Login with: admin / admin

You can maintain the Dolibarr and Letodms databases with phpMyAdmin (Login: root / your_root_password)
Homepage: http://www.letodms.com
ProjectLibre

ProjectLibre is a project management software system. It intends to be a complete desktop replacement for Microsoft Project.
Homepage: http://www.projectlibre.org
